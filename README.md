ЖАНГИРОВ ОСТАНОВИСЬ ЖАНГИРОВ ОСТАНОВИСЬ ЖАНГИРОВ ОСТАВИСЬ

Джентельменский набор, колеки:

```cmake
add_compile_options(
        -fdiagnostics-color=always
        -pedantic-errors
        -Werror
        -Wall
        -Wextra
        -Wpedantic
        -Wcast-align
        -Wcast-qual
        -Wconversion
        -Wctor-dtor-privacy
        -Wduplicated-branches
        -Wextra-semi
        -Wenum-compare
        -Wfloat-equal
        -Wnon-virtual-dtor
        -Woverloaded-virtual
        -Wredundant-decls
        -Wsign-conversion
)
```
